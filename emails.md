# EMAILS

### WORDS FOR EMAILS
| **FORMAL** | **EMAIL** |
|--------|-------|
| firstly | *to begin with* |
| secondly | *in addition* |
| thirdly | *further more* |
| lastly | *in conclusion* |
| We would like to tell you | *We would like to inform you* |
| We are not happy with the services | *We are not satisfied with the services* |
| Please dont forget | *I would like to remind you* |
| Should I send it this by EOD | *Would you like me to send it to you by end of the day* |
| Awaiting for a quick revert | *I would appreciate for a prompt response on this* |
| I am sorry for the delay | *I regret for the delay* |
